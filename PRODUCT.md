# LinguaLoop

## Problem

I use AI to write and understand English, but I forget the corrections afterward.

## Core promise

Help me write better English messages, understand English in context, and remember my recurring mistakes.

## MVP workflows

1. Message Coach
2. Reading Context Coach
3. Memory
4. Review

## Non-goals for MVP

- Admin dashboard
- Rich text editor
- Tiptap
- Worker queue
- Multi-provider AI
- Payment
- Browser extension
