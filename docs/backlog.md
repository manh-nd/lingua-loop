# Lingua Loop Backlog

This backlog captures useful ideas that are intentionally outside Phase 1. Product direction and phase scope live in [Product Direction](./product.md).

## Rich Workspace

- WYSIWYG or Notion-like editor.
- Inline accept/reject suggestions.
- Rich paste handling for email, Markdown, and documentation.
- Better diff visualization.
- Workspace search and saved drafts.

## Memory Retrieval and Personalization

- Embedding-based Memory retrieval.
- Semantic matching for mistakes expressed with different wording.
- Personalized learning profile built from saved MemoryItems and repeated corrections.
- Stronger ranking by relevance, recency, frequency, and practice performance.

## Reading to Practice

- Turn a Reading Coach result into an active writing task.
- Write a reply to the source text.
- Summarize the source text in English.
- Explain the source text to a teammate.
- Start a Live Coach roleplay from the reading context.

## Live Coach Expansion

- Memory-aware speaking sessions.
- Speaking Practice for due MemoryItems.
- Roleplays for standup, client updates, interviews, negotiation, incident explanation, and technical discussion.
- Better post-call learning reports.
- More guided speaking modes.

## Analytics and Insights

- Trend summaries for repeated mistakes.
- Practice completion and improvement history.
- Saved phrase and vocabulary usage patterns.
- Lightweight progress reporting if it helps the learning loop.

## Integrations and Distribution

- Browser extension.
- Native mobile app.
- Team or collaboration features.
- Payment and subscription.
- Public marketing site.

## Long-Document Capabilities

- Long-document knowledge base.
- Multi-document ingestion.
- Citations and source libraries.
- Enterprise knowledge search.
